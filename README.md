# DevOps_Pro1
Spring Boot REST API projesi - Azure üzerinde sanal makineye PostgreSQL ve Nginx reverse proxy ile deployment

## Proje Hakkında
Bu proje Spring Boot kullanılarak geliştirilmiş bir REST API uygulamasıdır(Hasta kayıt-randevu sistemi).
Uygulama Azure VM üzerinde yayınlanmış ve PostgreSQL veritabanı ile entegre edilmiştir.
Nginx reverse proxy kullanılarak 80 portu üzerinden erişim sağlanmıştır.

## Kullanılan Teknolojiler
- Java 21
- Spring Boot
- Maven
- PostgreSQL
- Nginx
- Azure VM

## Özellikler
- CRUD işlemleri (Create, Read, Update, Delete)
- Cloud veritabanı bağlantısı
- Nginx reverse proxy konfigürasyonu
- Maven ile JAR paketleme
